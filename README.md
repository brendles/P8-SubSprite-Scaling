# P8-SubSprite-Scaling
sspr() with controls to simulate scaling.




Basic controls using UP/DOWN/LEFT/RIGHT/Z/X


	UP:Minimize
	DOWN:Maximize
	LEFT:Right
	RIGHT:Left
	Z:Ascend
	X:Descend
